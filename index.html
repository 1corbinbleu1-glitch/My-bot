Privacy Policy

Application: Volos.ua Instagram Analytics Tool
Owner: Volos.ua (Oleksandra)
Last updated: July 2026

1. Overview

This application is a private, single-owner tool that connects to Meta's Instagram Graph API to retrieve business analytics data (posts, reels, stories, account insights) for the sole purpose of internal business reporting. The application is not available to the general public and does not collect data from or about any third-party users.

2. Data Collected via Instagram API

The application accesses the following data from the owner's own Instagram professional account using permissions granted through Meta's Instagram API with Instagram Login:

  • Basic account information (username, follower count, media count)
  • Media objects (posts, reels, stories) and their metadata (type, timestamp, caption)
  • Media-level insights (reach, plays, likes, saves, shares, comments)
  • Account-level insights (impressions, reach, profile views, website clicks)

No data from followers, commenters, or any other Instagram users is stored or processed.

3. How Data Is Used

All retrieved data is used exclusively for:

  • Generating internal performance reports for the account owner
  • Identifying content trends to inform the owner's publishing strategy

The data is never sold, shared with third parties, used for advertising, or transferred outside the owner's private infrastructure.

4. Data Storage

Analytics data is stored in a local SQLite database on a private server controlled solely by the account owner. Access tokens are stored in encrypted environment variables and are never logged or exposed.

5. Data Retention

Data is retained for as long as it is needed for business reporting purposes. The owner may delete all stored data at any time.

6. User Rights

Because this application accesses only the owner's own account and no third-party user data is processed, there are no third-party data subjects. The owner has full control over all data and may revoke the application's access at any time via Instagram Settings → Apps and Websites.

7. Meta Platform Compliance

This application is built on Meta's Instagram Graph API and complies with Meta's Platform Terms and Developer Policies. Instagram data is used only as permitted by those terms.
